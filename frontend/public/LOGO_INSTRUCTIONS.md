# Logo Instructions

## Current Logo

✅ Logo is already set up at: `Frontend/public/logo.jpg`

The app is using your logo file successfully!

## How to Update Your Logo

1. Replace the existing `logo.jpg` file in this folder (`Frontend/public/`)
2. The logo should be:
   - **Format**: JPG or PNG with transparent background (recommended)
   - **Size**: 500x500px or larger (square)
   - **Quality**: High resolution

## Current Logo Path

The app looks for the logo at: `/logo.jpg`

This means the file is at: `Frontend/public/logo.jpg`

## If Logo is Missing

If you want to change the logo:
1. Replace the file at `Frontend/public/logo.jpg`
2. Or add a new file and update the path in `Frontend/src/views/MenuView.vue` (line with `src="/logo.jpg"`)

## Logo Specifications

The logo is displayed as:
- **Width**: 48px
- **Height**: 48px
- **Shape**: Circular (border-radius: 50%)
- **Background**: White padding
- **Location**: Top-left of hero section
